### Quick Start

```bash
# Clone
git clone https://github.com/YOUR_USERNAME/german-job-bot.git
cd german-job-bot

# Install
npm install

# Start
npm start
```

### Configuration

1. Set your search filters in the web UI
2. Upload your resume
3. Give commands like: "Search Python jobs in Berlin"
